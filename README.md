# Buchun Mountain Blog

Welcome to the Buchun Mountain blog! This is a bilingual (Chinese & English) personal blog built with Astro.

## Features

- 🌐 **Bilingual Support**: Full i18n support with Chinese (default) and English.
- 🎨 **Minimal & Clean**: Based on the Astro blog template but optimized for personal use.
- 🚀 **Fast Performance**: 100/100 Lighthouse score.
- 📱 **Responsive**: Works great on mobile and desktop.
- 🔍 **SEO Optimized**: Sitemap, RSS feed, and meta tags included.

## Project Structure

- `src/content/blog/zh/`: Chinese blog posts.
- `src/content/blog/en/`: English blog posts.
- `src/i18n/`: Translation files and utilities.
- `src/pages/`: Page routes (including `en/` subfolder for English).

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
